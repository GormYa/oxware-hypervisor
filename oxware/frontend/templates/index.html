SSH ile Şifre Sıfırlama Yalnızca Yönetici

Şifreyi değiştirmek için SSH ile sunucuya root olarak bağlan ve aşağıdaki komutu çalıştır. Servis yeniden başlatıldığında uygulanır ve dosya silinir.

# Adım 1 — Sıfırlama dosyasını oluştur (root)
printf 'USERNAME=yeni_kullanici\nPASSWORD=yeni_sifre\n' > /etc/oxware/.passwd_reset
chmod 600 /etc/oxware/.passwd_reset
# Adım 2 — Servisi yeniden başlat
systemctl restart oxware

Dosya yalnızca root (uid=0) tarafından oluşturulduğunda işlenir. Servis uyguladıktan sonra otomatik siler.

Hosting Eklentileri

WiseCP ve WHMCS otomasyon modülleri — API üzerinden VM oluştur, sil, başlat, durdur.

Provisioning API Anahtarı

Billing sisteminiz bu anahtarı kullanarak OXware'e bağlanır. Anahtarın provisioning yetkisine sahip olması gerekir.

✅ API Anahtarı (yalnızca bir kez gösterilir — kopyalayın!):
Provisioning API Endpoints

Billing modülü bu endpointlere X-API-Key header ile istek atar. Sunucu adresi olarak OXware'in URL'sini, API anahtarı olarak yukarıda ürettiğiniz değeri kullanın.

POST /api/provision/create — VM oluştur
DELETE /api/provision/{vm_id} — VM sil
POST /api/provision/{vm_id}/suspend — VM durdur (suspend)
POST /api/provision/{vm_id}/unsuspend— VM başlat (unsuspend)
GET /api/provision/{vm_id}/status — VM durumu + IP
PUT /api/provision/{vm_id}/resize — CPU/RAM/Disk değiştir
GET /api/provision/templates — Mevcut OS template listesi
WiseCP Modülü

WiseCP'de Sunucular → Yeni Sunucu → OXware seçin.
Hostname = OXware URL, Password/API Key alanına API anahtarını girin.

Kurulum yolu:
modules/server/oxware/oxware.php
WHMCS Modülü

WHMCS'de Kurulum → Sunucular → Yeni Sunucu → OXware seçin.
Hostname = OXware URL, Access Hash = API anahtarı.

Kurulum yolu:
modules/servers/oxware/oxware.php
Hızlı Kurulum
  1. Yukarıdan modül dosyasını indirin.
  2. Billing panelinizde ilgili dizine yükleyin.
  3. Billing panelinde yeni sunucu oluşturun; Hostname = bu OXware'in adresi.
  4. API anahtarı oluşturun (sol üstteki kutu) ve Access Hash / API Key alanına yapıştırın.
  5. Billing panelinde Bağlantı Test butonu ile kontrol edin.
  6. Ürününüzü bu sunucuya bağlayın; CPU/RAM/Disk seçeneklerini girin.
